## How to use

Install it and run in development mode:

```bash
yarn
yarn start
```

Install it and build:

```bash
yarn
yarn build
```

Install it and bundle in production mode:

```bash
yarn
yarn bundle
```

Install it and bundle in analyze mode:

```bash
yarn
yarn analyze
```

Run integration tests:

```bash
yarn test
```

Format code:

```bash
yarn format
```
